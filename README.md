# blobd

[AT Protocol Blob](https://atproto.com/specs/data-model#blob-type)-serving HTTP Server in Go

## Installation

```bash
go install github.com/atscan/blobd@latest
```

## Usage

```bash
blobd -d /path/to/data -p 3000
```
